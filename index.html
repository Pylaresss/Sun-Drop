About the Project

Solar Powered Smart Irrigation System for Sustainable Agriculture

Agriculture is one of the most important sectors worldwide, yet it faces major challenges such as water scarcity, climate variability, and the need to optimize energy consumption. Traditional irrigation systems often rely on manual operation or fixed schedules, which can lead to inefficient water usage and unnecessary energy consumption.

This project proposes a solar-powered smart irrigation system designed to provide a sustainable and energy-efficient solution for small-scale agriculture. The system combines renewable energy, environmental sensors, and IoT technologies to automate irrigation based on real-time environmental conditions.

System Overview

The irrigation system is powered by a photovoltaic solar panel that charges a battery used to supply energy to the entire system. A microcontroller (ESP32) collects data from several sensors installed in the environment.

  • Soil moisture sensor
  • Temperature and humidity sensor
  • Ultrasonic sensor to monitor water tank level
  • Energy monitoring sensors

Using this information, the system determines when irrigation is necessary and automatically activates a water valve to irrigate the field.

Smart Irrigation Decision

Unlike traditional irrigation systems, this solution integrates a decision logic based on environmental data and weather forecasts.

  • Soil humidity level
  • Current temperature
  • Water level in the tank
  • Weather forecast for the next 24 hours

For example, if rainfall is expected, the system can recommend delaying irrigation to avoid wasting water.

IoT Monitoring Platform

All sensor data is transmitted to the cloud using the ESP32 Wi-Fi connectivity. The data is stored and visualized using ThingSpeak, allowing users to monitor the irrigation system remotely.

A custom web dashboard was developed to display:

  • Soil moisture level
  • Water tank level
  • Temperature
  • Irrigation status
  • Weather forecast
  • Irrigation recommendations
